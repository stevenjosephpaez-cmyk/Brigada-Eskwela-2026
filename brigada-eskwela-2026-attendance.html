📋 How to Use

  1. I-host ang HTML file sa GitHub Pages, Netlify, o kahit local network.
  2. Buksan ang QR Code tab sa projector o print-out.
  3. I-scan ng mga kalahok gamit ang kanilang camera/QR app.
  4. Pupunta sila sa Attendance tab at pupunan ang form.
  5. Pumunta sa Admin tab → i-download ang CSV para sa Excel.
⚠️ Note: Para gumana ang QR code, dapat naka-host ang file sa isang accessible na URL (hindi lang local file path). Gamitin ang Network URL kapag naka-live server.
